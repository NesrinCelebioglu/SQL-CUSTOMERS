# Customers Database SQL Projekt
## Projektbeschreibung

Dieses Projekt wurde entwickelt, um SQL-Abfragen und Datenanalysefähigkeiten anhand einer Kundendatenbank zu verbessern. Die Daten aus einer Excel-Datei wurden in einer Datenbanktabelle namens Customers importiert, und verschiedene Abfragen wurden zur Analyse der Kundendaten durchgeführt.

### Projektziel

- Erlernen von SQL-Abfragen zur Filterung, Sortierung, Gruppierung und Verwendung von Joins.

- Analyse von Kundendaten basierend auf Alter, Stadt, Geschlecht, Telefonanbieter und anderen Kriterien.

- Übung mit Subqueries und Joins.

- Hinzufügen neuer Spalten zur Datenbanktabelle und Aktualisierung von Daten.

### Inhalt

Im Rahmen dieses Projekts wurden die folgenden Aufgaben durchgeführt:

- Erstellung der Datenbank Customers und Import der Daten aus der Excel-Datei.

- Filtern von Kunden nach ihrem Namen (z. B. Namen, die mit dem Buchstaben „A“ beginnen).

- Filtern nach Geburtsjahr (einschließlich 1990 und 1995).

- Abfragen von Kunden basierend auf der Stadt unter Verwendung von Joins.

- Abrufen von Kunden aus einer Stadt mithilfe von Subqueries.

- Analyse der Anzahl von Kunden pro Stadt.

- Auflisten von Städten mit mehr als 10 Kunden, sortiert nach der Kundenzahl von hoch nach niedrig.

- Analyse der Anzahl männlicher und weiblicher Kunden pro Stadt.

- Hinzufügen einer neuen Spalte AGEGROUP in der Tabelle Customers.

- Aktualisierung der Spalte AGEGROUP nach Altersgruppen (20–35, 36–45, 46–55, 55–65 und 65+ Jahre).

- Auflisten von Kunden in Istanbul, deren Bezirk nicht Kadıköy ist.

- Extrahieren der Telefonanbieterinformationen aus den Telefonnummern (TELNR1 und TELNR2).

- Analyse der Kundenanzahl pro Anbieter basierend auf der Telefonnummer.

- Auflisten der Bezirke mit den meisten Kunden pro Stadt, sortiert von hoch nach niedrig.

- Anzeigen der Wochentage der Geburtstage der Kunden (Montag, Dienstag, …).

### Projektziele

- Verbesserung der SQL- und Datenanalysefähigkeiten.

- Erfahrung in Datenbankdesign und Tabellenmanagement.

- Simulation von praxisnahen Analysen von Kundendaten.
